# docusaurus

## Installation

```bash
npm install @lite-tree/react
// or
yarn add @lite-tree/react
// or
pnpm add @lite-tree/react

```

## Usage

Import `@lite-tree/react` in `.mdx` file.

```mdx
import { LiteTree } from "@lite-tree/react"
<LiteTree data={`
Root
    A
        A1
        A2
    B
        B1
        B2
`}/>
```
