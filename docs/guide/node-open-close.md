# 展开/折叠

`LiteTree`组件支持通过`+`和`-`前置符号来控制节点的默认展开/折叠状态。

```ts
<LiteTree>
根节点
    + 节点1                     // 默认展开状态
        - 节点1.1
        - 节点1.2
    - 节点2                     // 默认折叠状态
        - 节点2.1   
        - 节点2.2
</LiteTree>
```

**渲染效果如下：**

<LiteTree>
根节点
    + 节点1                     // 默认展开状态
        节点1.1
        节点1.2
    - 节点2                     // 默认折叠状态
        节点2.1
        节点2.2
</LiteTree>
